# Reverse-Engineered Dictionaries

This repository contains dictionaries that were originally published as PDFs and have been reverse-engineered into tidy, structured datasets.

All included dictionaries are publicly available and distributed under licenses that allow information sharing.

⚠️ **Disclaimer:** While care was taken during the conversion process, errors may have been introduced and some information may have been missed. Always consult the original sources when using this data for research or publication.

## Sources

- **Kadiwéu** — Griffiths, Glyn. *Dicionário da língua Kadiwéu: Kadiwéu–Português, Português–Kadiwéu.* (2002).
